# 金手指设定素材库

> 每条素材必须满足：核心能力明确 + 限制机制完整 + 可直接用于创作
> 入库门槛：综合评分 ≥ 7/10

---

## GF-001 · 时间回溯
- **类型**：异能
- **核心能力**：可回溯3秒时间，回到3秒前的自身状态和位置
- **激活条件**：精神力消耗，本能触发（危险时自动激活）或主动触发
- **限制机制**：
  - 每次使用后24小时内运气减半（因果律反噬）
  - 连续使用间隔不得少于1小时，否则精神力枯竭
  - 回溯只影响自身，无法改变已发生的客观事件（别人受的伤、打碎的物品不变）
  - 回溯距离随等级提升：3秒→10秒→30秒→1分钟→3分钟
- **成长路径**：回溯时长延长 + 可携带物品回溯 + 可部分回溯（只回溯身体不回溯记忆）
- **适用类型**：都市异能 / 悬疑推理 / 末世生存
- **来源**：原创
- **评分**：9/10（独特性9 + 可操作性9 + 扩展性9 + 类型契合度8）
- **扩展方向**：
  - 变体A：只能回溯情感状态（回到某时刻的心境）
  - 变体B：可回溯物体状态（修复破损物品）
  - 变体C：回溯记忆（回到过去重新体验某段记忆并发现遗漏细节）

## GF-002 · 万物合成
- **类型**：系统/异能
- **核心能力**：可将两件物品合成为新物品，合成结果由物品属性和合成规则决定
- **激活条件**：双手接触两件物品，心念触发
- **限制机制**：
  - 合成失败率随物品等级递增（普通物品5%→稀有物品30%→传说物品80%）
  - 合成失败则两件物品都消失
  - 每日合成次数有上限（初始3次，随等级提升）
  - 无法合成活物
  - 合成需要消耗"合成点数"，点数通过分解物品获得
- **成长路径**：合成槽位增加（2件→3件→4件）+ 成功率提升 + 可合成活物 + 可合成抽象概念
- **适用类型**：玄幻 / 系统流 / 都市异能
- **来源**：原创
- **评分**：8/10（独特性7 + 可操作性9 + 扩展性9 + 类型契合度8）
- **扩展方向**：
  - 变体A：只能合成同类物品（武器+武器=更强武器）
  - 变体B：合成结果有随机性（可能出意外好物/废品）
  - 变体C：可合成技能（两种功法融合）

## GF-003 · 体质共享
- **类型**：血脉/契约
- **核心能力**：可与灵兽建立契约，共享灵兽的体质属性（力量/速度/感知/特殊能力）
- **激活条件**：与灵兽达成契约（需灵兽自愿），首次契约消耗大量精神力
- **限制机制**：
  - 同时承受灵兽受到的伤害（灵兽受伤，契约者同等比例受伤）
  - 契约数量有上限（初始1只，随等级提升最多3只）
  - 契约解除条件苛刻，强行解除双方都受重创
  - 灵兽死亡，契约者永久损失对应比例的体质
  - 共享比例：初期30%，随契约深化提升至70%
- **成长路径**：契约数量增加 + 共享比例提升 + 可反向强化灵兽 + 契约融合（多只灵兽属性叠加）
- **适用类型**：修仙 / 玄幻 / 都市异能
- **来源**：原创
- **评分**：8/10（独特性8 + 可操作性8 + 扩展性8 + 类型契合度9）
- **扩展方向**：
  - 变体A：可与非生物建立契约（与宝剑共享锋利，与铠甲共享坚固）
  - 变体B：可窃取他人灵兽的属性（暗黑路线）
  - 变体C：灵兽进化后解锁全新属性类型

## GF-004 · 记忆继承
- **类型**：异能/血脉
- **核心能力**：可继承死去之人的部分记忆，获得其技能、知识、甚至情感片段
- **激活条件**：接触死者遗体或重要遗物，且死者死去不超过7天
- **限制机制**：
  - 每次继承会覆盖自身部分记忆（随机的，可能覆盖重要记忆）
  - 继承的记忆会逐渐消退（7天内完整，30天内模糊，90天后只剩碎片）
  - 继承过多会导致人格混乱（不同人的记忆和性格打架）
  - 无法主动选择继承哪些记忆（随机抽取）
  - 继承死者的执念会被一同继承，可能影响行为
- **成长路径**：继承时间窗口延长 + 可选择性增强 + 记忆保持时间延长 + 可继承更多深层记忆
- **适用类型**：都市异能 / 悬疑推理 / 修仙
- **来源**：原创
- **评分**：9/10（独特性10 + 可操作性9 + 扩展性8 + 类型契合度9）
- **扩展方向**：
  - 变体A：只能继承临死前3秒的记忆（推理向）
  - 变体B：可继承活人的记忆（但对方会失忆）
  - 变体C：继承到某种程度可以"召唤"死者意识短暂附体

## GF-005 · 概率操纵
- **类型**：异能/规则
- **核心能力**：可感知事件的发生概率，并微调概率值
- **激活条件**：精神力集中，对目标事件进行"概率观测"后再"概率微调"
- **限制机制**：
  - 调整幅度不超过±15%（如原概率30%可调至15%-45%）
  - 消耗精神力与调整幅度成正比
  - 同一事件只能调整一次
  - 无法影响已经确定的事件（因果律锁定）
  - 过度使用会导致"概率反噬"——自己遭遇低概率坏事的概率上升
- **成长路径**：调整幅度提升（±15%→±25%→±40%）+ 可连续调整 + 可感知更远未来的概率
- **适用类型**：都市异能 / 科幻未来 / 悬疑
- **来源**：原创
- **评分**：8/10（独特性9 + 可操作性7 + 扩展性8 + 类型契合度8）
- **扩展方向**：
  - 变体A：只能感知概率不能调整（纯信息型金手指）
  - 变体B：可将自身"好运气"储存，关键时刻一次性释放
  - 变体C：概率操纵的范围可扩大到影响他人

## GF-006 · 言灵之力
- **类型**：血脉/规则
- **核心能力**：说出的某些关键词可触发对应效果（如"破"可震碎眼前物体，"定"可短暂定住目标）
- **激活条件**：口头说出，必须发音清晰、意图明确
- **限制机制**：
  - 每个言灵词有冷却时间（"破"冷却1小时，"定"冷却3小时）
  - 言灵效果与使用者修为/精神力挂钩，修为不足则效果极弱
  - 说出的谎言会反噬自身（言灵使用者不能说谎，否则力量受损）
  - 每次使用言灵消耗寿命（微量但累积可观）
  - 言灵词数量有限（初始3个，随领悟增加）
- **成长路径**：言灵词数量增加 + 效果增强 + 冷却缩短 + 可组合言灵（先"定"再"破"）
- **适用类型**：玄幻 / 修仙 / 都市异能
- **来源**：原创
- **评分**：9/10（独特性9 + 可操作性9 + 扩展性9 + 类型契合度9）
- **扩展方向**：
  - 变体A：言灵必须用古语/特定语言
  - 变体B：写下文字也有同样效果（文灵）
  - 变体C：对方重复你的言灵词可被你控制（危险的双刃剑）

## GF-007 · 命格改写
- **类型**：规则/命运
- **核心能力**：可看到他人的"命格"（命运走向的简化图谱），并可用自身"命格值"改写他人命运节点
- **激活条件**：与目标对视3秒以上，即可看到对方命格
- **限制机制**：
  - 改写他人命运需要消耗自身等量的"命格值"
  - 命格值通过自身经历命运转折时获取（越大的转折获取越多）
  - 无法改写自己的命格
  - 改写后的命运会产生蝴蝶效应，可能比原来更糟
  - 每次改写后7天内不能再改写同一人
- **成长路径**：命格值上限提升 + 可改写更重要的命运节点 + 可看到更远未来的命格
- **适用类型**：都市异能 / 悬疑 / 历史
- **来源**：原创
- **评分**：9/10（独特性10 + 可操作性8 + 扩展性9 + 类型契合度9）
- **扩展方向**：
  - 变体A：只能改写，不能看到（凭直觉改写）
  - 变体B：命格值可交易（多人之间的命运经济）
  - 变体C：改写到一定程度可以"编写"全新命格

## GF-008 · 灵魂出窍
- **类型**：修仙/异能
- **核心能力**：灵魂可脱离肉体独立行动，穿墙、远视、感应灵气，肉身留在原地
- **激活条件**：深度冥想状态，需3分钟准备
- **限制机制**：
  - 灵魂离体期间肉身毫无防御，受到攻击灵魂也会受损
  - 灵魂离体有时间限制（初始30分钟，超时灵魂开始消散）
  - 灵魂状态无法接触实物，只能观察
  - 某些阵法/结界可困住灵魂
  - 灵魂离体期间肉身会逐渐虚弱
- **成长路径**：离体时间延长 + 灵魂可轻微影响实物 + 可附身无主之体 + 灵魂攻击能力
- **适用类型**：修仙 / 玄幻 / 悬疑
- **来源**：原创（修仙类经典设定改编）
- **评分**：7/10（独特性6 + 可操作性8 + 扩展性7 + 类型契合度8）
- **扩展方向**：
  - 变体A：可附身动物（借动物之躯行动）
  - 变体B：灵魂可进入他人梦境
  - 变体C：多个灵魂可合体形成更强的灵魂体

## GF-009 · 等价炼成
- **类型**：规则/系统
- **核心能力**：可将等价的事物进行转化——付出什么代价，获得什么等价的结果
- **激活条件**：明确说出"等价交换"的双方（"我用XX换取XX"）
- **限制机制**：
  - 必须严格等价（系统判定，无法欺骗）
  - 交换不可逆
  - 每次交换有10%的"损耗"（付出110%获得100%）
  - 无法交换生命（不能"用我的寿命换他人的复活"）
  - 交换过程中无法行动，容易被打断
- **成长路径**：损耗率降低 + 可交换更多类型 + 可交换抽象概念（用记忆换力量）+ 可进行不等价交换（需特殊权限）
- **适用类型**：玄幻 / 都市异能 / 修仙
- **来源**：原创（炼金术等价交换原理改编）
- **评分**：8/10（独特性8 + 可操作性9 + 扩展性8 + 类型契合度8）
- **扩展方向**：
  - 变体A：可暂缓交付（先获得后付费，但利息极高）
  - 变体B：可为他人做等价交换（中间商赚差价）
  - 变体C：不同领域的"价值"换算率不同，可套利

## GF-010 · 知识灌输
- **类型**：系统/异能
- **核心能力**：可通过触摸书籍/卷轴/任何信息载体，瞬间获取其中的全部知识
- **激活条件**：双手接触信息载体，保持3秒
- **限制机制**：
  - 获取的知识不等于理解（知道公式不等于会用）
  - 一次灌输过多知识会导致头痛、精神混乱
  - 获取的知识有时间衰减（7天内完整，逐渐遗忘至基础印象）
  - 无法从加密/有灵力保护的信息载体中获取
  - 每日使用次数有上限（初始5次）
- **成长路径**：日使用次数增加 + 知识保持时间延长 + 可理解更深层的知识 + 可从活人身上"读取"知识
- **适用类型**：都市重生 / 修仙 / 历史 / 科幻
- **来源**：原创
- **评分**：8/10（独特性7 + 可操作性9 + 扩展性8 + 类型契合度9）
- **扩展方向**：
  - 变体A：只能获取某种类型的知识（如只能获取战斗技能）
  - 变体B：获取知识时会体验原作者的情感
  - 变体C：可将自身知识"写入"空白载体

## GF-011 · 模拟器
- **类型**：系统
- **核心能力**：可以模拟未来人生走向，在模拟中经历完整人生轨迹，但每次模拟消耗寿元/精神力，模拟结果可带回现实部分记忆或能力碎片
- **激活条件**：闭眼进入冥想状态，主动触发模拟；首次使用需消耗10点精神力作为"启动代价"
- **限制机制**：
  - 每次模拟消耗一年寿元，模拟时间越长消耗越大（模拟一百年消耗十年寿元）
  - 模拟中的记忆带回现实时会模糊化，只保留关键节点的印象和情绪，细节大量遗失
  - 模拟中获得的能力碎片仅保留10%的威力，且需在现实中重新修炼才能巩固
  - 连续模拟间隔不得少于7天，否则精神力枯竭导致昏迷三天
  - 模拟结果并非唯一未来，只是基于当前状态的"最可能走向"，重大变故可能导致模拟结果完全偏离
- **成长路径**：模拟时长延长 + 记忆保留率提升（10%→30%→50%→80%）+ 能力碎片保留比例增加 + 可在模拟中尝试不同选择形成"周目经验"
- **适用类型**：都市 / 玄幻 / 诸天流
- **来源**：原创
- **评分**：8/10（独特性8 + 可操作性8 + 扩展性9 + 类型契合度8）
- **扩展方向**：
  - 变体A：模拟的不是自己的未来，而是他人的——可模拟目标人物的过去和未来，但消耗双倍寿元
  - 变体B：模拟中死亡会遭受真实的精神创伤（痛感100%真实），多次"模拟死亡"可能导致心理崩溃
  - 变体C：可将模拟中的"遗憾"带回现实化为执念之力，遗憾越深力量越强但越难以控制

## GF-012 · 规则感知
- **类型**：异能/感知
- **核心能力**：可感知当前区域的隐藏规则（如规则怪谈中的生存规则、秘境中的行为禁忌），以模糊的直觉或片段化文字形式呈现
- **激活条件**：进入新区域时自动触发浅层感知，深度感知需主动集中精神力凝视区域核心
- **限制机制**：
  - 感知到的规则可能不完整——只感知到"不能做某事"但不告知违反后果，或只感知到规则的前半段
  - 感知结果可能有误导——某些"规则"会主动伪装，让你感知到的是假规则或陷阱规则
  - 高等级规则感知消耗大量精神力，连续深度感知三个区域后精神力枯竭
  - 无法感知"规则制定者"的身份和位置，只能感知规则本身
  - 感知到的规则无法直接用语言告知他人（一出口就变模糊），只能自己理解体会
- **成长路径**：感知准确率提升（50%→70%→90%）+ 可感知更复杂的规则 + 可辨别真伪规则 + 可感知规则之间的关联与逻辑链
- **适用类型**：规则怪谈 / 无限流 / 悬疑
- **来源**：原创
- **评分**：9/10（独特性9 + 可操作性9 + 扩展性9 + 类型契合度10）
- **扩展方向**：
  - 变体A：只能感知"禁止类"规则，无法感知"允许类"规则——知道什么不能做但不知道什么可以做
  - 变体B：感知到的规则可以写入"规则笔记"记录并分享给他人，但每次分享自身感知力永久衰减5%
  - 变体C：感知规则到极致可"预测"规则变化趋势，在规则潮汐前提前布局抢占先机

## GF-013 · 基建面板
- **类型**：系统/面板
- **核心能力**：获得类似游戏的建筑建造面板，可消耗资源建造/升级领地建筑，面板提供蓝图和效率加成，但建筑需真实资源与时间完成
- **激活条件**：拥有至少一处属于自己的领地（哪怕是一间破屋），系统自动激活
- **限制机制**：
  - 面板只提供蓝图和建造效率加成（建造速度+50%，资源消耗-20%），不凭空产生资源
  - 每次建造需消耗真实材料（石料、木材、金属等），面板无法无中生有
  - 高级建筑蓝图需要完成特定条件解锁（如"建过10座普通建筑"解锁中级蓝图）
  - 建造过程中如果资源中断，建筑会停工且已投入资源不退回
  - 面板每日使用次数有上限（初始5次，随领地等级提升）
- **成长路径**：蓝图等级提升 + 建造效率加成增加 + 可建造建筑类型扩展 + 解锁"自动建造"功能
- **适用类型**：基建流 / 种田 / 领主
- **来源**：原创
- **评分**：8/10（独特性7 + 可操作性9 + 扩展性8 + 类型契合度9）
- **扩展方向**：
  - 变体A：面板可建造"超自然建筑"——如聚灵阵、传送门，但需要超自然材料
  - 变体B：建造速度与领地居民数量挂钩——人越多建造越快，但需养活所有人
  - 变体C：面板可"拆解"敌方建筑获取蓝图碎片，集齐碎片可复制敌方建筑

## GF-014 · 直播系统
- **类型**：系统
- **核心能力**：行为被直播到异界/全网，观众打赏可转化为实力提升（灵气、修为、技能经验等），打赏越多提升越快

## GF-023 · 历史修正

- **成长路径**：锻造成功率提升（40%→60%→80%）+ 可锻造更复杂的灵魂改造 + 锻造疼痛减轻 + 解锁"灵魂融合"功能
- **适用类型**：修仙 / 玄幻 / 灵魂系
- **来源**：原创
- **评分**：8/10（独特性9 + 可操作性7 + 扩展性8 + 类型契合度8）
- **扩展方向**：
  - 变体A：只能锻造武器型灵魂——将灵魂锻造成灵魂兵器，但锻造者自身灵魂也化为武器无法再拥有肉身
  - 变体B：锻造失败的灵魂不会消失而是变成"残魂"，残魂可能反过来侵蚀锻造者
  - 变体C：可将多个灵魂碎片锻造成全新灵魂体（如人造灵兽灵魂），但新灵魂可能产生不可控的自我意识

## GF-025 · 异界任务板
- **类型**：系统/任务
- **核心能力**：可接取异界发布的任务，完成任务获得跨世界的奖励（功法、材料、特殊能力等），任务类型涵盖战斗、探索、收集、护送等
- **激活条件**：系统自动激活，每日刷新任务列表
- **限制机制**：
  - 任务失败有惩罚——轻则扣除实力/资源，重则强制执行"惩罚任务"（难度远超原任务）
  - 任务难度不可预知——任务描述只有简短信息，真正的难度和隐藏条件需接取后才能发现
  - 每次只能同时进行一个任务，放弃任务等同于失败
  - 部分任务有隐藏的"道德代价"——看似简单的任务可能要求牺牲无辜者
  - 任务截止时间严格，超时一秒也算失败
- **成长路径**：可同时接取任务数增加 + 任务信息更详细 + 可拒绝任务而不受惩罚 + 解锁"自选任务"功能
- **适用类型**：诸天流 / 都市 / 无限流
- **来源**：原创
- **评分**：8/10（独特性7 + 可操作性9 + 扩展性8 + 类型契合度9）
- **扩展方向**：
  - 变体A：任务的发布者可以看到执行者的实时状态——你在做任务时发布者可能在"观战"
  - 变体B：完成足够多的任务后可以成为"发布者"——自己设计任务让别人执行，获取执行者的部分成果
  - 变体C：任务板存在"隐藏任务"——在特定条件（如濒死、特定时间、特定地点）才会显示
