# 商业库索引

> 市场定位与商业策略素材——"怎么卖"的方法论

## 读者心理 (reader-psychology)

### 代入感来源
| 类型 | 核心代入机制 | 设计要点 |
|---|---|---|
| 玄幻修仙 | 草根逆袭的代入 | 主角出身要低但不自卑，读者看到自己 |
| 都市重生 | "如果重来一次"的代入 | 前世遗憾要具体，读者能共情 |
| 系统流 | 游戏化成长的代入 | 系统界面/数值/等级要让玩家感到熟悉 |
| 悬疑推理 | 好奇心驱动 | 每个谜题都要让读者觉得自己"快想到了" |
| 末世生存 | 生存本能的代入 | 压力要真实，资源要紧张 |

### 追读驱动力
| 驱动力 | 机制 | 设计方法 |
|---|---|---|
| 爽感驱动 | 爽点密度和强度 | RT-001/003 节奏模板 + SP系列爽点配方 |
| 悬念驱动 | 信息缺口 | FB系列伏笔 + 章末钩子 |
| 情感驱动 | 角色情感羁绊 | CH系列角色 + IS-004/007情感名场面 |
| 好奇驱动 | 世界观/设定悬念 | WV系列世界观 + 设定揭示节奏 |
| 竞争驱动 | 排名/对战/博弈 | LV系列升级体系 + CF系列冲突 |

### 社交传播点
| 传播类型 | 设计要点 | 示例 |
|---|---|---|
| 金句传播 | 精炼到可以截图的震撼表达 | "吾为天帝，当镇压世间一切敌" |
| 反转传播 | 让人忍不住分享的意外 | 以为的救兵是敌人的第二层布局 |
| 共情传播 | 触发情感共鸣的细节 | 母亲在主角不知道的时候默默付出 |
| 设定传播 | 独特到让人想讨论的设定 | 诡秘之主的22条途径 |

## 付费转化 (conversion-strategies)

### 免费章节策略
| 类型 | 推荐免费章数 | 转换关键 |
|---|---|---|
| 玄幻修仙 | 20-30章 | 在第一次大升级/大揭示前设付费点 |
| 都市重生 | 15-20章 | 在"第一桶金"高潮后设付费点 |
| 系统流 | 10-15章 | 在系统升级/解锁新功能前设付费点 |
| 悬疑推理 | 8-12章 | 在第一个谜题即将揭示前设付费点 |
| 无限流 | 10-15章 | 在第一个副本即将通关前设付费点 |

### 日更策略
| 节奏 | 每日字数 | 更新时间 | 适用类型 |
|---|---|---|---|
| 高频快更 | 6000-8000字/天 | 早中晚三更 | 快节奏爽文 |
| 标准日更 | 4000-6000字/天 | 晚间单更 | 大部分类型 |
| 精品周更 | 15000-20000字/周 | 固定时间 | 慢热/悬疑/文青 |
