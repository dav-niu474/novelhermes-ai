## 📝 使用方式

创作时引用素材ID即可，例如：
- "使用 GF-012 的规则感知 + LV-010 的规则怪谈评级"
- "参考 SP-017 规则碾压的爽点配方设计高潮"
- "用 OP-006 规则宣读开头 + CH-009 规则猎手的人物模板开篇"
- "查拆文素材 GF-D001"（访问 analysis/deconstruction-materials/golden-fingers/）

Agent 会从对应 catalog.md 中读取完整素材内容。
拆文素材使用 XX-DNNN 格式ID，存放在 analysis/deconstruction-materials/ 下。
