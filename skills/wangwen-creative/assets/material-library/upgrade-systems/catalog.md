# 升级体系素材目录

> 本目录收录网文创作中可复用的升级体系模板，每条体系包含等级划分、突破条件、战力锚点、体系天花板等核心要素。
>
> 使用原则：①战力锚点必须具象化，杜绝"很强""极强"等模糊描述；②突破条件需兼具内在修炼与外在触发；③等级跨越的节奏须与章节节奏匹配。

## LV-001 九境修仙

| 字段 | 内容 |
|------|------|
| **ID** | LV-001 |
| **体系名称** | 九境修仙 |
| **来源** | 原创（融合古典修仙体系与道教内丹学说） |
| **适用类型** | 仙侠、修真、玄幻、洪荒流 |

### 等级划分

| 境界 | 对应章节 | 战力锚点 |
|------|----------|----------|
| 淬体 | 1–50章 | 拳可裂碑，腿可断木，身受刀剑仅留白痕；百米冲刺仅需三息，但连续激战半个时辰则力竭 |
| 凝气 | 50–120章 | 可御气飞行不过百丈高度，初窥神识方圆十丈；一掌击出可碎石屋，但须蓄力三息，且灵气耗尽需打坐恢复至少两个时辰 |
| 筑基 | 120–220章 | 遁速可达音速三成，神识覆盖方圆里许；可祭出法器远攻，飞剑百步取人首级，然法力仅支撑三场全力战斗 |
| 金丹 | 220–350章 | 一剑可削平百丈山头，但丹田金丹裂则修为尽散；寿元五百载，可神游太虚不过三日；法力涌动间令方圆十里飞沙走石 |
| 元婴 | 350–500章 | 元婴出窍可瞬移千里，但肉身无人守护则如待宰羔羊；怒意一放可令方圆百里生灵跪伏，然每动用一次本源则寿元折损十年 |
| 化神 | 500–650章 | 可小范围改写天地法则（如逆转重力、冻结时间三息），但每改一条法则需闭关十年消化反噬；一念可覆盖千里感知 |
| 合体 | 650–800章 | 肉身与天地合一，寻常攻击穿体而过如清风拂面；可开辟方圆百里的独立空间，但维持此空间需持续消耗本源，至多支撑一日 |
| 大乘 | 800–950章 | 可召唤天劫劈敌，自身几近不朽——被灭肉身后以一滴精血即可重聚，需时三月；言出法随，一语可令万里晴空变雷海 |
| 渡劫 | 950–1100章 | 渡过天劫则飞升，渡不过则形神俱灭；巅峰战力可碎方圆百里，但三月内功力减半；可窥见天道一角，却无法逆转生死大限 |

### 突破条件

- **淬体→凝气**：需引天地灵气入体，开十二正经与奇经八脉；若无灵脉辅助，普通人需苦修三年，有灵脉则三月可成
- **凝气→筑基**：需筑基丹一枚+三日夜不眠打坐引气归元；灵气稀薄之地突破概率不足一成，灵气浓郁之地可达五成
- **筑基→金丹**：须领悟一门道法真意（非死记硬背），于丹田凝出金丹；悟性差者终身卡于此境，悟性绝佳者可一朝顿悟
- **金丹→元婴**：金丹九转，须经历心魔劫——心魔以最深执念化形，过不去则走火入魔、修为倒退一境
- **元婴→化神**：须观天地法则运行并融于己身，需至少观摩一次天劫全程（可旁观他人渡劫），且有足够悟性提炼法则碎片
- **化神→合体**：须将领悟之法则与肉身熔炼，过程凶险，失败则法则反噬、肉身崩碎、元婴重伤，需至少百年恢复
- **合体→大乘**：须渡"情劫"——非战斗型劫难，需在红尘中了却因果牵绊，斩不断则永远困于合体巅峰
- **大乘→渡劫**：天地自行降下天劫，不可选择时机，九重雷劫依次落下，每重间隔一日；可准备但不可回避

### 体系天花板

飞升上界后本体系不再适用，需转修上界功法；飞升前为当前位面最强存在，但仍受天道规则约束——不可逆转已死之人的生死（此为天道底线，触之必遭天罚降世）。

### 评分

| 维度 | 分数 | 说明 |
|------|------|------|
| 完整度 | 9/10 | 九境环环相扣，逻辑自洽 |
| 可扩展性 | 10/10 | 可嵌套上界体系、横向加入魔修/妖修分支 |
| 读者友好度 | 8/10 | 经典框架易上手，但后期境界描写难度大 |
| 创新性 | 6/10 | 属于经典范式，创新空间在细节填充 |
| **综合** | **8.3/10** | |

### 扩展方向

1. **魔修分支**：同九境但突破条件为"杀戮证道"，心魔劫改为"嗜血反噬"，每突破一境需一条性命为祭
2. **妖修变体**：以化形为关键节点，渡劫改为渡"人劫"——需斩断兽性方能飞升
3. **双修体系嵌套**：男女主角各修不同分支，关键时刻可联手施出合体技，战力临时跃升两境但事后双双重伤
4. **残缺修仙**：灵气枯竭时代的修仙，每境只修一半便需强行突破，战力打折但突破速度快一倍

## LV-002 三阶异能觉醒

| 字段 | 内容 |
|------|------|
| **ID** | LV-002 |
| **体系名称** | 三阶异能觉醒 |
| **来源** | 原创（都市异能经典范式提炼） |
| **适用类型** | 都市异能、末世、都市热血、校园异能 |

### 等级划分

| 阶段 | 细分等级 | 对应章节 | 战力锚点 |
|------|----------|----------|----------|
| **觉醒期** | 初觉 | 1–40章 | 异能刚显现，不可控——火系可能打喷嚏就烧了眉毛，冰系一紧张就冻住自己的手；战斗中异能触发率约三成，威力仅相当于普通人用工具（如火柴、水枪） |
| | 半觉 | 40–80章 | 异能可在情绪稳定时主动触发，但持续时间极短（火系可持续燃烧十秒，冰系可凝出拳头大的冰块）；可勉强用于实战，但过度使用后头痛欲裂、鼻血直流 |
| | 全觉 | 80–130章 | 异能完全觉醒，可稳定操控——火系可凝出火球持续燃烧一分钟，冰系可冻结一扇门；但异能使用有冷却期，连续全力释放三次后异能暂时失效，需休息至少半个时辰 |
| **掌控期** | 微控 | 130–200章 | 可精细操控异能——火系可只在指尖燃一簇烛火，也可爆出丈许火墙；冰系可凝出冰刃做武器，也可铺一层薄冰封路；冷却期缩短至全力释放五次 |
| | 精控 | 200–300章 | 异能如臂使指，可同时操控多个异能节点——火系可同时在十个位置点火爆袭，冰系可制造冰雕分身；全力释放次数可达十次，但精神力耗尽后会昏睡一天 |
| | 极控 | 300–400章 | 可将异能压缩至极致——火系压缩火球可熔穿钢板，冰系凝出的冰硬度堪比金刚石；可异化出副属性（火系→雷电，冰系→雾气），但副属性威力仅为主属性三成 |
| **领域期** | 伪域 | 400–500章 | 方圆十丈内异能效果倍增，敌人进入此范围则被压制三成实力；但领域维持需全神贯注，无法同时攻击 |
| | 真域 | 500–650章 | 方圆百丈为绝对领域，领域内可改写物理规则（火域内水会蒸发、冰域内金属变脆）；可边维持领域边攻击，但领域边缘不稳，高手可强行突破 |
| | 神域 | 650–800章 | 方圆里许为神域，领域内可凭意志创造或毁灭物质（火域可凭空燃起火海、冰域可冻结时间三息）；领域一出则同级之下无人可挡，但施展一次神域需休养七天 |

### 突破条件

- **觉醒期各细分**：无特殊条件，随异能自然成长；但生死危机可加速觉醒——濒死体验可使初觉直接跳至半觉，但代价是异能永久不稳定（有10%概率失控暴走）
- **全觉→微控**：需在实战中对异能有"手感"上的领悟，单纯修炼无效；需至少经历十场实战并存活，且至少三次将异能用到极限
- **精控→极控**：需经历一次异能失控危机——过度使用导致异能反噬，在反噬中领悟异能本质；若失败则异能永久降级一个细分等级
- **极控→伪域**：需领悟异能与空间的关联，非修炼可达，多为机缘触发——如在异能共鸣的特殊地点冥想、或目睹他人展开领域后顿悟
- **真域→神域**：需将自身意志与异能彻底融合，过程如涅槃——需主动引爆自身异能再重新凝聚，失败则异能尽失沦为凡人，成功概率仅两成

### 体系天花板

神域为都市位面极限，再往上则异能开始影响区域天象（如火系神域者长期存在会导致方圆百里干旱），天道意志会自动压制超出上限的异能——强行突破天花板者将被"天罚"抹杀。

### 评分

| 维度 | 分数 | 说明 |
|------|------|------|
| 完整度 | 9/10 | 三阶九级，节奏清晰 |
| 可扩展性 | 8/10 | 可叠加异能类型变种，但领域期之后扩展空间有限 |
| 读者友好度 | 9/10 | 都市背景+异能，读者代入感极强 |
| 创新性 | 7/10 | 框架经典，细分等级和突破条件有新意 |
| **综合** | **8.3/10** | |

### 扩展方向

1. **异能变异体系**：觉醒后异能可能变异——火系变异为灰烬（腐蚀性）、冰系变异为霜魂（精神攻击），变异方向不可控但战力通常更强
2. **双异能冲突**：极少数人同时觉醒两种异能，但两种异能互相排斥，使用一种时另一种会自动压制，唯有领域期才可能融合
3. **异能剥夺与移植**：科学阵营可剥离异能核心移植他人，但排异反应概率高达六成，失败则双方异能尽毁
4. **觉醒事件调查线**：异能觉醒并非偶然，背后有组织在人为触发觉醒事件，主角追查真相贯穿全书

## LV-003 诡秘序列体系

| 字段 | 内容 |
|------|------|
| **ID** | LV-003 |
| **体系名称** | 诡秘序列体系 |
| **来源** | 改编自《诡秘之主》（爱潜水的乌贼） |
| **适用类型** | 诡秘、克苏鲁、维多利亚风、悬疑推理、西幻 |

### 等级划分

| 序列 | 对应章节 | 战力锚点 |
|------|----------|----------|
| 序列9 | 1–60章 | 刚服食魔药，获得单一超凡能力（如"占卜家"可进行纸牌占卜，准确率约六成；"小丑"可控制面部表情和身体柔韧度）；能力弱小但有质变——已非普通人，但一个持枪壮汉仍可威胁性命 |
| 序列8 | 60–130章 | 能力大幅强化且增加分支（"占卜家"→"小丑"，占卜准确率升至八成+可操控自身灵体丝线进行微操）；可正面击败三五个普通人，但面对热武器仍需谨慎 |
| 序列7 | 130–220章 | 质变节点——能力产生飞跃（"魔术师"可进行火焰跳跃等瞬间移动，但每次跳跃距离不超过五十米且需有火焰媒介）；可与全副武装的小队抗衡，但会被军队围杀 |
| 序列6 | 220–320章 | 获得领域性能力（"无面人"可完美变成任何见过的人，包括声音、气息、甚至部分超凡能力）；单人可渗透任何普通组织，正面战力相当于一支精锐小队 |
| 序列5 | 320–420章 | 能力质变为概念级（"秘偶大师"可将活人变成受操控的秘偶，但需在对方不防备时以灵体丝线侵入，且同时操控上限为三具）；一人可覆灭一个小型组织，但正面对抗军队仍显吃力 |
| 序列4 | 420–530章 | 半神门槛——可封神，拥有神性与人性交织（"诡法师"可将自身存在隐藏于历史与现实的夹缝中，极难被杀死）；可在城市级别造成影响，但神性过强会导致人性流失，需定期"锚定"人性 |
| 序列3 | 530–640章 | 可操控区域级规则（可令一定范围内的物理法则偏移）；一人可对抗一支军队，但神性侵蚀加剧，每次动用神性后人性会进一步削弱 |
| 序列2 | 640–760章 | 接近天使——可小范围改写现实（言出法随的雏形，但每改写一次需消耗自身灵性储备，且改写内容不可与现有天使的权柄冲突）；可毁灭一座城市但会引来其他序列2的围攻 |
| 序列1 | 760–880章 | 天使级——可长驻某一权柄领域（如"时间"领域内可暂停时间五息、回溯时间三息）；存在本身即是对低序列的绝对压制，序列5以下无法直视其真容 |
| 序列0 | 880–1000章 | 真神——权柄领域扩展至概念本身（"愚者"权柄可令一切针对自身的攻击"被愚弄"而偏离）；但不死不灭仅限于权柄领域内，离开领域则仍有弱点；且真神不可直接干涉物质界，需通过代言人行事 |

### 突破条件

- **核心机制——魔药消化**：服食对应序列魔药后，必须通过"扮演"魔药名称对应的角色来消化魔药——消化不完全则可能失控变异为怪物，消化完美则突破顺利且副作用最小
- **仪式辅助**：序列4以上突破需特定仪式（如"诡法师"突破需在满月之夜于特定地点完成"欺骗历史"的仪式），仪式失败则魔药反噬，轻则疯癫，重则异变为怪物
- **魔药配方获取**：每个序列的魔药配方都是核心资源，获取方式包括：从神秘组织购买、自行研究推演、从古代遗迹中发现；配方的稀缺性直接决定突破速度
- **失控风险**：每次服食魔药都有失控概率，序列越高概率越大（序列9约5%，序列4约40%，序列1约70%），失控后理智崩塌、肉身异变，成为无意识的恐怖怪物

### 体系天花板

序列0真神为当前体系天花板，但真神之上传闻有"旧日"——超越序列的不可名状存在，真神在其面前亦如蝼蚁；旧日不可被杀死，只能被封印，且封印会随时间衰减。

### 评分

| 维度 | 分数 | 说明 |
|------|------|------|
| 完整度 | 10/10 | 22条路径×10个序列，体系庞大且逻辑自洽 |
| 可扩展性 | 10/10 | 可自创新路径、新魔药、新仪式，扩展性极强 |
| 读者友好度 | 7/10 | 体系复杂，读者入门门槛较高 |
| 创新性 | 9/10 | 魔药消化+扮演机制极具创新性 |
| **综合** | **9.0/10** | |

### 扩展方向

1. **东方诡秘路径**：设计适配东方文化的序列路径——如"阴阳师"路径（序列9风水先生→序列8画符师→序列7阴阳眼→…→序列0司命）
2. **魔药交易黑市线**：以魔药配方为核心资源设计经济体系，主角可通过买卖配方积累资源
3. **失控者社群**：失控变异者并未全部失去理智，部分保留了部分人性，形成地下社群——他们是最了解失控真相的人
4. **跨路径融合**：极少数人尝试融合两条路径的魔药，结果可能是前所未有的力量，也可能是瞬间毁灭性的失控

## LV-004 商业帝国阶梯

| 字段 | 内容 |
|------|------|
| **体系名称** | 商业帝国阶梯 |
| **来源** | 原创（都市商战题材经典升级范式） |
| **适用类型** | 都市商战、重生流、系统流、职场逆袭 |

### 等级划分

| 等级 | 对应章节 | 战力锚点（商业影响力） |
|------|----------|----------------------|
| 街头 | 1–40章 | 兜里不足千元，靠摆地摊、代购、跑腿赚第一桶金；可调动资源仅自身劳动力+极少量启动资金；一单生意利润不过百元，但已有商业嗅觉萌芽 |
| 地摊 | 40–90章 | 日流水破千，拥有固定摊位或线上小店；可雇佣一两个帮手；单笔交易利润可达数百；开始理解进货、定价、周转的逻辑，但经不起任何政策风浪 |
| 小店 | 90–160章 | 拥有实体店铺或稳定线上店铺，月流水十万级；雇佣五至十人团队；品牌名开始在本地小范围传播；可承受一次小型商业失误（约损失数万），但连续两次失误则回退至地摊级 |
| 连锁 | 160–260章 | 同城开出三至五家分店，月流水百万级；核心团队二十人以上；开始建立供应链关系，可谈批量进货折扣；但仍受制于区域性竞品和地方政策，一次重大决策失误可能导致半数门店关闭 |

### 突破条件

- **街头→地摊**：赚到第一个一万元并持续盈利三个月——看似简单，但需要从"用时间换钱"转变为"用钱换时间"的思维跃迁
- **地摊→小店**：积累五万以上启动资金+找到可复制的盈利模式；关键转折点通常是发现一个"信息差"——别人不知道的进货渠道或客群需求
- **小店→连锁**：建立标准化运营手册（SOP），使单店模式可复制；需招募并培养出至少两个可独当一面的店长；突破的真正难点不在资本而在"信任"——能否放权给他人经营
- **连锁→集团**：需完成一次关键并购或拿到一笔战略融资；此阶段突破的核心是"从经营者转变为资本运作者"——不再亲自管业务，而是管人和管钱；若无法完成思维转变则永远卡在连锁级
- **集团→财阀**：需建立至少一条完整产业链（从上游原材料到下游零售），形成行业护城河；关键突破点通常是"打败一个老牌财阀"或"抓住一次时代级风口"（如互联网转型、新能源革命）
- **财阀→帝国**：需完成跨国布局+建立金融闭环（自己的银行或基金）；此阶段突破已非商业能力可决定，需要政治资源、国际视野和至少一次"孤注一掷"的全押——赢了封神，输了归零

### 体系天花板

帝国级为商业体系天花板，再往上则进入政治领域——资本与权力的博弈超越商业逻辑；帝国级商人虽富可敌国，但在国家级力量面前仍需低头（如反垄断调查、国有资产征收等）。

### 突破条件

- **灵气浓度门槛**：每个阶段的突破都要求世界灵气浓度达到对应区间——不是个人修炼够就能突破，而是"世界没准备好，你就不可能突破"；这意味着前期所有人的突破节奏是同步的
- **初觉→感灵**：灵气浓度达标后，需在一次灵气潮汐中主动引导灵气冲击识海——成功则灵识开启，失败则昏迷三天但无永久损伤
- **感灵→御灵**：需找到与自己属性契合的灵气节点（如火山口、深海、雷暴中心）进行深度修炼，在节点处修炼一日等于外界百日；但节点通常被灵兽或势力占据
- **御灵→通灵**：须经历一次"灵气共鸣"——在特定的天地异象中（如日食、灵气潮汐顶峰）与天地灵气产生共振，共振成功则突破；共振失败则经脉碎裂，需至少一年恢复
- **通灵→化灵**：须将肉身浸泡在极高浓度灵气中（仅灵气泉眼或SS级灵脉可达）七日夜不间断，期间承受灵气改造肉身的剧痛；中途退出则前功尽弃且经脉受损
- **化灵→归灵**：需在灵气浓度达到60%的节点处，主动放弃肉身抵抗让灵气"吞噬"自身——若与灵气的融合度足够则重生为归灵，若融合度不足则形神俱灭，无人知道自己的融合度是多少

### 体系天花板

归灵为灵气复苏体系天花板，但归灵者并非无敌——归灵者依赖天地灵气，若灵气被截断则战力暴跌；且灵气浓度达到100%后世界将进入"大净化"——所有灵气将重新归零循环，归灵者需在净化中保住修为或选择渡劫飞升。

### 突破条件

- **布衣→秀才**：须通过县试、府试、院试三级考试——考的是八股文功底+临场发挥+考官眼缘；若文章出众可连过三级，若有一级落榜则需等三年再考
- **秀才→举人**：须通过乡试（省试），录取率约3%——考的不只是文采，还有对时务的理解；可"买举人"（捐纳），但买来的举人在官场被歧视
- **举人→进士**：须通过会试+殿试，录取率约1%——殿试由皇帝亲自出题，考的是对帝王心思的揣摩；进士名次直接决定起点（一甲入翰林、二甲可入部、三甲外放）
- **进士→知县**：正式入仕，需等"铨选"——吏部根据缺额分配官职，有人等三年有人等三天；若有人脉可"运作"到富庶之县，无人脉则可能被发配穷乡僻壤
- **知县→知府**：三年一次"大计"考核——考核优秀者可升迁，但"优秀"的标准不只看政绩，还要看与上司的关系、送礼的分量、站队的正确性；纯靠政绩升迁者不足两成
- **知府→巡抚**：须获朝中重臣保举+政绩卓著+站对派系——三者缺一不可；此阶段突破的核心已非能力而是"政治资本"，若无靠山则终身卡在知府
- **巡抚→尚书**：须入京任职（通常先任侍郎再升尚书）+在朝堂党争中站对最终赢家；此阶段突破需要"政治嗅觉"——提前判断哪位皇子会继位、哪个派系会掌权，赌错了可能陪葬
- **尚书→丞相**：无固定路径——须同时满足：帝王信任+百官服众+军权支持+无致命把柄；历史上能同时满足这四项的人寥寥无几，且即使坐上丞相之位，维持时间平均不超过五年

### 突破条件

- **核心机制——界力积累**：穿越每个世界需消耗界力，界力通过完成世界核心任务获得；界力不足则无法穿越下一世界，只能停留当前世界继续积累

- **因果积累**：在之前世界留下的因果会追随穿越者——你在凡人世界结的仇敌可能转世到仙神世界成为强者找你报仇；因果越重，在新世界的起点越低
- **回归限制**：已穿越的世界可以回归，但回归时实力被压制至该世界天花板水平，且每次回归消耗额外界力

### 体系天花板

本源世界为体系终极天花板，但到达本源世界后穿越者面临终极选择——成为新世界的"创世者"（永远留在一个新世界中作为造物主），或是成为"永恒行者"（继续穿越但永远无法停下），两条路都无法回头。

| C级·规则者 | 60–140章 | 已理解规则的表层逻辑，可利用规则间的矛盾和漏洞求生；进入C级规则域存活率约六成；能在规则允许的范围内找到"最优行动路径"，但遇到规则突变则束手无策 |
| B级·规则制定者 | 140–240章 | 可在局部范围内"制定"临时规则（如在某个房间内设定"此处禁止杀戮"），但制定的规则持续时间有限且受环境规则压制；进入B级规则域存活率约七成；可主动引导他人遵守或违反规则 |
| A级·领域之主 | 240–360章 | 可展开"规则领域"——领域内自身制定的规则优先级高于环境规则；领域半径约50米，持续时间约1小时；进入A级规则域存活率约八成；可感知规则的"诞生"和"消亡"节点 |

### 突破条件

- **D→C**：需在一次规则域中主动利用规则漏洞存活——不是靠运气躲过，而是靠理解规则逻辑主动创造生存空间；需至少经历5个不同规则域并总结出规则运作的通用模式
- **C→B**：需亲手"制定"第一条规则——在生死关头以精神力强行烙印一条规则到空间中，成功率约三成；失败则精神力受损需恢复一个月，且该规则域对自身永久敌视
- **B→A**：需理解规则的"本源逻辑"——不是"规则是什么"而是"规则为什么是这样"，需在一次规则突变中捕捉到规则诞生的瞬间并与之共鸣；共鸣失败则被规则反噬，评级永久降至C
- **A→S**：需找到规则的"制定者"——每个规则域背后都有一个规则制定者（可能是人、可能是存在、可能是自动机制），找到它并与之对话或对抗；此突破无固定路径，取决于制定者的性质

### 体系天花板

S级规则之源为当前体系天花板，但传闻中存在"SS级·规则虚无者"——能够"取消"规则的存在，让一切规则归零。但这只是传说，因为取消规则等于取消秩序，可能引发规则域的全面崩溃。

### 扩展方向

1. **规则寄生线**：某些规则会"寄生"在评级者身上，赋予额外能力但逐渐侵蚀人格——评级越高寄生越深
2. **规则对抗赛**：B级以上评级者之间的对决不是比武力而是比规则制定——在同一区域内各自制定规则，规则冲突时以逻辑更自洽者获胜
3. **规则创世线**：S级评级者可尝试创造一个全新的规则域，从零制定所有规则——但这等于创造一个小世界

功德金仙为体系天花板，但功德金仙面临终极悖论——"功德是否是另一种形式的业力？"当你执着于积累功德时，这种执着本身就是一种"业"。功德金仙的最终考验可能是"放下功德"。

### 评分

| 维度 | 分数 | 说明 |
|------|------|------|
| 完整度 | 9/10 | 六级递进+功德劫机制，逻辑自洽 |
| 可扩展性 | 8/10 | 可叠加善恶抉择线/业力对路线 |
| 读者友好度 | 8/10 | 功德值直观，但道德哲学讨论门槛略高 |
| 创新性 | 8/10 | 功德修仙+功德劫设定有深度 |
| **综合** | **8.3/10** | |

### 扩展方向

1. **业力修士对路线**：以业力为力量的暗黑路线——杀戮越重业力越强，业力修士与功德修士天生对立
2. **善恶灰色线**：某些善行需要恶手段——杀一人救百人，这种"以恶行善"的功德计算方式成为剧情核心矛盾

| 推演者 | 50–130章 | 可推演剧本的隐藏线索和他人角色——通过观察行为推断他人身份和动机；可进行"逻辑推理"——消耗精神力模拟剧本走向，准确率约六成；开始影响剧本走向——在关键节点做出"非角色设定"的选择 |

- **导演→造物主**：需将一个真实区域（至少一个街区）转化为剧本场景——所有进入区域的人不知不觉成为角色；关键突破是"打破虚构与现实的边界"

### 体系天花板

造物主为体系天花板，但造物主面临"作者悖论"——当你的剧本完美到连自己都被困在其中，你还能分清自己是作者还是角色吗？造物主的终极考验是"走出自己的剧本"。

| 创新性 | 9/10 | 将剧本杀转化为升级体系极具创意 |
| **综合** | **9.0/10** | |

### 扩展方向

1. **剧本杀对决线**：两个导演级以上的人各自编排剧本对撞——两套叙事逻辑碰撞，更精妙的一方获胜

| **适用类型** | 修仙、玄幻、灵魂系、灵魂锻造 |

### 等级划分

| 等级 | 对应章节 | 战力锚点 |
|------|----------|----------|
| 学徒 | 1–50章 | 可感知灵魂的存在——看到灵魂的轮廓和颜色；可进行最基础的灵魂修复——愈合灵魂上的小裂痕；锻造成功率约40%，失败则灵魂原料报废；每次锻造后需休息一天 |
| 匠人 | 50–130章 | 可锻造灵魂赋予新属性——如增强灵魂的韧性、赋予灵魂火焰抗性等；锻造成功率约60%，可同时处理两个灵魂；解锁"灵魂鉴定"——可精确评估灵魂的品质、属性和潜在能力 |
| 大师 | 130–240章 | 可进行"灵魂重塑"——彻底改变灵魂的结构和属性，如将普通灵魂重塑为战斗型灵魂；锻造成功率约80%，解锁"灵魂共鸣"——与锻造对象的灵魂产生共鸣，获得灵感加成；但共鸣过程中自身灵魂可能被对方影响 |
| 宗师 | 240–380章 | 可锻造"灵魂器"——将灵魂锻造成可装备的物品，如灵魂之剑、灵魂之盾；灵魂器拥有独立意识，可与使用者对话和配合；锻造成功率约90%，但宗师级锻造每次消耗自身灵魂的一部分 |

### 突破条件

- **匠人→大师**：需完成一次"灵魂重塑"——将一个完全不同类型的灵魂转化为目标类型，成功率仅三成；失败则自身灵魂受损
- **大师→宗师**：需锻造出第一件"灵魂器"——将灵魂锻造成有形的器物，需要同时掌握灵魂锻造和器物锻造两种技艺
- **宗师→神匠**：需在灵魂锻造中达到"与道合一"的境界——不再需要工具和仪式，灵魂本身即是锻造炉；突破条件不明，历史上仅有三人达到此境界

### 体系天花板

神匠为体系天花板，但神匠创造灵魂的能力引出了一个终极问题——创造出的灵魂是否有"人权"？如果造物主可以随意毁灭自己的造物，那与暴君何异？神匠的终极考验是"对自己造物的责任"。

| 完整度 | 8/10 | 五级递进清晰，但后两级突破条件可再细化 |
| 可扩展性 | 8/10 | 灵魂类型无限，可叠加灵魂市场/灵魂战争线 |
| 读者友好度 | 7/10 | 灵魂锻造概念稍抽象，需具象化描写 |
| 创新性 | 8/10 | 灵魂锻造体系较独特，灵魂器设定有亮点 |
| **综合** | **8.0/10** | |

### 扩展方向

1. **灵魂黑市线**：非法的灵魂交易市场——贩卖他人灵魂、强夺灵魂碎片，灵魂匠人的暗面
2. **灵魂战争线**：不同灵魂匠人阵营之间的战争——以灵魂器为武器的战斗
3. **灵魂进化线**：被锻造的灵魂可能自主进化，超出锻造者的控制——造物超越造物主

## LV-020 时空行者等级

| 字段 | 内容 |
|------|------|
| **ID** | LV-020 |
| **体系名称** | 时空行者等级 |
| **来源** | 原创（时空穿越/历史修正题材经典升级范式提炼） |
| **适用类型** | 科幻、时空穿越、历史修正、悬疑 |

### 等级划分

| 等级 | 对应章节 | 战力锚点 |
|------|----------|----------|
| 观察者 | 1–50章 | 可"观测"历史事件的真相——回到过去但不参与，如同看一场全息电影；观测有范围限制——只能观测自身所在位置方圆百米内的历史；每次观测消耗精神力，过度观测会导致"时间眩晕" |
| 干预者 | 50–130章 | 可在历史中做出微小干预——与历史人物对话、移动物品、留下痕迹；但干预幅度严格受限——只能改变"不影响历史大势"的细节；每次干预会在时间线上留下"因果伤痕"，伤痕被时空监管者追踪 |
| 修正者 | 130–240章 | 可修正历史偏差——当历史偏离"应然走向"时，修正者可将其拨回正轨；修正的规模随等级提升——从修正一个人的命运到修正一场战争的走向；但每次修正消耗自身存在感（被越来越多人遗忘），修正规模越大消耗越多 |
| 管理者 | 240–380章 | 可管理一段历史时间线——确保该时间线上的事件按"应然走向"运行；管理者有"时间权限"——可以暂停、快进、回放自己管理的时间段；但管理者的时间线可能与其他管理者的时间线冲突，冲突时需进行"时间谈判" |
| 掌控者 | 380–550章 | 可掌控整个时间线的走向——不再受"应然走向"约束，而是自己决定时间线的方向；掌控者可以"创造"新的时间线分支，也可以"剪除"已有的分支；但掌控者有一个绝对禁忌——不可消除自身存在的时间线根源，否则自身将从所有时间线中消失 |

### 突破条件

- **观察者→干预者**：需在一次观测中"忍不住"干预——看到悲剧即将发生而主动出手；第一次干预无论成功与否都标志突破，但干预的后果需要自行承担
- **干预者→修正者**：需成功修正一次历史偏差——找到偏差的源头并消除，让历史回归正轨；修正失败则偏差固化，该时间线永久偏移
- **修正者→管理者**：需被"时空委员会"（管理所有时间线的组织）认可并授予管理权限；认可标准不明，但已知修正历史偏差的规模和精准度是核心指标
- **管理者→掌控者**：需在管理时间线的过程中发现"时间线的终极真相"——所有时间线的源头和终点是什么；发现真相后可以选择成为掌控者或选择"退出"（放弃一切时空能力回归普通人）

### 体系天花板

掌控者为体系天花板，但掌控者面临"时间悖论"——如果你掌控了所有时间线，那你的过去、现在和未来都是你自己的选择，自由意志是否还存在？掌控者的终极困境是"选择的意义"。

### 评分

| 维度 | 分数 | 说明 |
|------|------|------|
| 完整度 | 9/10 | 五级递进逻辑严密，权限递增清晰 |
| 可扩展性 | 9/10 | 时间线无限，历史事件无限，扩展空间极大 |
| 读者友好度 | 7/10 | 时空概念稍复杂，需要读者跟随思考 |
| 创新性 | 9/10 | 从观察到掌控的权限递进+存在感消耗机制极具创新性 |
| **综合** | **9.0/10** | |

### 扩展方向

1. **时空犯罪线**：有组织专门利用时空能力犯罪——偷取历史文物、暗杀历史人物、篡改历史记录
2. **时间线战争线**：不同阵营的管理者争夺同一条时间线的控制权，战争不是武力对决而是"时间线争夺"
3. **时间尽头线**：所有时间线是否有一个共同的终点？到达终点的掌控者会发现什么？
4. **存在感回收线**：被遗忘的存在感是否可以回收？如果可以，那些被遗忘的人是否还能回来？
