# 悬疑探案 · 类型模板

> "真相只有一个，但谎言有千百种。"

## 类型定位

- **核心卖点**：逻辑推演+悬念层层+真相反转+智力碾压
- **读者画像**：20-40岁，喜欢推理和智斗，享受"猜不到结局"的快感
- **创作难度**：⭐⭐⭐⭐⭐（伏笔布局是灵魂，逻辑链一断全盘崩）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 超强推理 | 大脑运算力超群，可快速推理 | 推理基于已知信息，信息不足就会误判 | 侧写能力/逻辑推演/犯罪心理学 |
| 灵异感知 | 可感知死者残留信息/灵异线索 | 信息模糊如碎片，需要拼凑解读 | 通灵/读残念/灵视/阴阳眼 |
| 时间回溯 | 可短暂回溯案发时间线 | 回溯有时间/次数限制，且有盲区 | 回溯24小时/触碰物体回溯/梦中回溯 |
| 因果追踪 | 可追踪事件因果链条 | 只能看到直接因果，间接因果需要自己推导 | 因果之眼/命运之线/时间之沙 |

## 默认升级体系

悬疑探案的"升级"不是数值增长，而是推理能力和案件层级的提升：

```
小案初探（1-30章）· 普通刑事案件
  → 连环迷局（30-80章）· 跨区域/跨时间系列案
    → 幕后组织（80-150章）· 牵出背后庞大势力
      → 棋局博弈（150-250章）· 与幕后黑手的智斗
        → 真相漩涡（250-400章）· 真相牵扯更深的秘密
          → 终局揭秘（400+章）· 揭开所有谜团
```

**升级设计要点**：
- 每个案件应有关键线索3-5条，红鲱鱼2-3条
- 小案解决后要有"未尽之谜"连接下一个案件
- 真相要分层：表面真相→深层真相→最终真相
- 主角成长体现在：从被动破案到主动设局

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 案发开局 | 1-5章 | 案件发生，展示诡异之处 | 快速建立悬念，勾住读者 |
| 线索搜集 | 5-20章 | 深入调查，发现矛盾线索 | 层层推进，制造疑点 |
| 迷雾重重 | 20-35章 | 嫌疑人增多，每个都有嫌疑 | 节奏放缓，读者参与推理 |
| 关键突破 | 35-45章 | 发现关键证据/灵光一闪 | 高潮，逻辑链闭合 |
| 真相揭露 | 45-55章 | 揭露真相+凶手动机 | 情感冲击+逻辑满足 |
| 余波暗涌 | 55-65章 | 案件了结但留下更大悬念 | 铺垫下一个案件 |

## 开篇钩子推荐

### 钩子1：密室诡案（经典）
> 展示一个看似不可能的犯罪场景 → 所有线索都指向矛盾 → 到底是谁，怎么做到的？

### 钩子2：身份反转（紧张）
> 主角以嫌疑人身份登场 → 必须自证清白的同时找出真凶 → 双重压力下的推理

### 钩子3：连环预告（紧迫）
> 凶手提前预告下一次犯罪 → 主角必须在时限内破解预告 → 与时间的赛跑

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 天降线索 | 关键线索凭空出现，读者无法推理 | 所有线索前文必须铺垫，读者理论上有机会推理出真相 |
| 红鲱鱼滥用 | 每个嫌疑人都是假的，反转成习惯 | 红鲱鱼要有逻辑基础，真相的反转要让读者"恍然大悟"而非"被欺骗" |
| 破案靠金手指 | 主角的特殊能力直接看到真相 | 金手指提供碎片信息，推理过程必须靠主角智力完成 |
| 逻辑链断裂 | 前后矛盾，时间线对不上 | 每个案件写完整的时间线表和逻辑树，反复校验 |
| 案件割裂 | 一个案子接一个案子，没有主线 | 每个案件都要推进主线，用"未尽之谜"串联全局 |
