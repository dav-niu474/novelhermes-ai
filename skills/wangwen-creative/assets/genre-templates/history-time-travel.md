# 历史穿越 · 类型模板

> "以千年后之智，行千年之前之事。"

## 类型定位

- **核心卖点**：历史知识碾压+权谋博弈+改写历史+文化碰撞
- **读者画像**：20-40岁，有历史兴趣，享受"如果我在那个时代"的代入感和改变历史的爽感
- **创作难度**：⭐⭐⭐⭐（历史知识储备是门槛，权谋逻辑要自洽，改编尺度难把握）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 历史先知 | 知道这段历史的走向 | 蝴蝶效应改变后续历史，先知优势递减 | 全知型/模糊型/触发型 |
| 现代知识 | 掌握超越时代的科技/制度知识 | 缺乏工业基础，先进理念落地困难 | 科技知识/军事知识/经济知识/制度知识 |
| 随身空间 | 携带现代物资穿越 | 空间有限且不能补充 | 仓库空间/图书空间/制造空间 |
| 文气加身 | 穿越后获得文气/才气加持 | 才气来源于作品质量，不是自动获得 | 诗词立道/文章立命/书法通神 |

## 默认升级体系

历史穿越的"升级"是社会地位和影响力的跃升：

```
布衣草民（1-30章）· 一无所有，立足为艰
  → 小有声名（30-80章）· 获得基本社会认可
    → 地方豪强（80-150章）· 在一地拥有势力
      → 朝堂重臣（150-250章）· 进入权力核心
        → 一方诸侯（250-400章）· 割据一方或深度影响朝局
          → 天下共主（400+章）· 改写历史走向
```

**升级设计要点**：
- 每次跃升都要通过具体的"功绩"实现，而非空降
- 现代知识的运用要有边界：缺工业基础造不了蒸汽机
- 政治博弈要有深度：不是靠嘴炮，而是利益交换和势力平衡
- 蝴蝶效应：主角的每个选择都在改变历史走向，先知优势逐渐失效

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 穿越适应 | 1-5章 | 穿越到历史时期，适应身份 | 铺垫为主，展示时代特征 |
| 崭露头角 | 5-25章 | 用现代知识/历史先知获得第一次成功 | 爽点密集 |
| 树敌招忌 | 25-45章 | 引起既得利益者的警觉和打压 | 冲突升级 |
| 危机博弈 | 45-70章 | 在权力博弈中求存 | 智斗为主，节奏紧张 |
| 破局上位 | 70-100章 | 化解危机并获得更高地位 | 高潮爆发 |
| 格局升级 | 100+章 | 进入更大的权力舞台 | 世界观展开，博弈深化 |

## 开篇钩子推荐

### 钩子1：绝境穿越（经典）
> 穿越到即将被处决/灭门的历史人物身上 → 必须立刻自救 → 用现代思维化险为夷

### 钩子2：诗词惊世（文抄公流）
> 穿越到文道盛行的时代 → 脱口而出一首千古绝唱 → 一诗成名天下知

### 钩子3：先知布局（权谋流）
> 穿越到历史转折点前夕 → 知道即将发生什么 → 提前布局，等风来

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 历史硬伤 | 明显不符合历史常识 | 做好基础历史功课，关键节点核对史料 |
| 现代傲慢 | 古人全是傻子，主角降维打击 | 古人的智慧在他们的时代是合理的，给对手体面 |
| 文抄公滥用 | 抄诗词解决一切问题 | 诗词只是敲门砖，权力和利益才是核心博弈 |
| 工业化速成 | 几章之内就搞出工业革命 | 科技进步有前置条件，要循序渐进 |
| 后宫失焦 | 历史女性角色全爱上主角 | 给女性角色独立的身份和立场，不做附庸 |
| 政治幼稚 | 权谋就是杀人灭口 | 真实权谋是利益分配、制度设计、人心掌控 |
