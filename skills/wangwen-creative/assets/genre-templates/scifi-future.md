# 科幻未来 · 类型模板

> "文明的尽头，是另一种开始。"

## 类型定位

- **核心卖点**：科技想象+星际文明+社会变革+人类命运
- **读者画像**：18-35岁，喜欢硬核科技设定和宏大叙事，享受"人类何去何从"的哲学思辨
- **创作难度**：⭐⭐⭐⭐⭐（科技设定要有前瞻性且自洽，社会推演要有深度，叙事不能太空洞）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 科技图谱 | 脑中植入超级科技数据库 | 需要相应工业基础才能实现，跨级科技有危险 | 外星遗产/未来记忆/AI辅助 |
| 基因进化 | 基因改造获得超凡能力 | 改造有副作用，多次改造可能导致基因崩溃 | 超级战士/新人类/共生体 |
| AI共生 | 与超级人工智能共生 | AI有自己的意志，可能产生分歧 | 芯片植入/AI伙伴/意识上传 |
| 文明火种 | 掌握某种可重启文明的钥匙 | 启动条件苛刻，且会引来觊觎 | 方舟系统/星图/创世程序 |

## 默认升级体系

科幻未来的"升级"是文明层级的跃升，参照卡尔达舍夫等级但更细化：

```
行星级（1-50章）· 掌控母星全部资源
  → 恒星级（50-120章）· 利用整个恒星系资源
    → 星际级（120-220章）· 跨星系航行与殖民
      → 星域级（220-350章）· 掌控一个星域的文明网络
        → 宇宙级（350-500章）· 理解宇宙基本规律
          → 超越级（500+章）· 突破现有宇宙认知
```

**升级设计要点**：
- 每个层级对应不同的科技树、社会结构和认知维度
- 科技突破要有过程：理论→实验→应用→普及
- 社会随科技进步而变革：制度、伦理、人性都在重新定义
- 高层级文明的出现要有伏笔，不能突然跳出

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 时代变革 | 1-10章 | 科技突破引发社会巨变 | 铺垫世界观，建立代入感 |
| 个人崛起 | 10-30章 | 主角在变革中找到定位 | 爽点+悬念 |
| 星际初探 | 30-60章 | 首次接触外星文明/进入太空 | 格局跃升，新奇感拉满 |
| 文明碰撞 | 60-100章 | 不同文明的冲突与交流 | 冲突升级，智斗+武力 |
| 存存危机 | 100-150章 | 面对威胁文明的存续危机 | 高压叙事，节奏加快 |
| 宇宙真相 | 150+章 | 探索宇宙本质，哲学层面的追问 | 格局最大化，思辨性增强 |

## 开篇钩子推荐

### 钩子1：科技奇点（经典）
> 人类文明迎来重大科技突破 → 社会剧变 → 主角身处变革漩涡中心

### 钩子2：星际遗民（史诗）
> 人类文明已经扩张到星际 → 发现远古文明的遗迹 → 这个遗迹改变了人类的命运走向

### 钩子3：末日重生（紧张）
> 文明即将毁灭 → 主角回到关键节点 → 能否改变人类的命运？

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 设定堆砌 | 大段科技说明文，故事被设定淹没 | 科技设定融入剧情和人物，通过事件展示而非解释 |
| 科幻变玄幻 | 后期变成披着科幻皮的修仙 | 科技体系要有内部逻辑，不能靠"意志力"突破 |
| 外星人脸谱化 | 外星文明都是邪恶侵略者或善良导师 | 给外星文明独特的文化逻辑和价值观 |
| 忽视社会面 | 科技进步但社会结构没变化 | 科技改变生产方式→改变社会关系→改变人性 |
| 宇宙太大太空 | 星际尺度太大导致读者无感 | 用具体人物和故事锚定宏大叙事 |
