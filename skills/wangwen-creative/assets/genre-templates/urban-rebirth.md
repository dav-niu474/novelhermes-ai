# 都市重生 · 类型模板

> "重活一世，这世不留遗憾。"

## 类型定位

- **核心卖点**：先知优势+逆袭改命+弥补遗憾
- **读者画像**：25-40岁男性，有生活阅历，追求代入感和爽感
- **创作难度**：⭐⭐（好上手，但要写出深度需要功力）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 重生记忆 | 记住前世所有经历 | 未来因蝴蝶效应逐渐偏移 | 商业重生/学术重生/娱乐圈重生 |
| 重生+系统 | 重生并获得系统辅助 | 系统任务失败有惩罚 | 投资系统/学业系统/社交系统 |
| 重生+空间 | 重生并获得随身空间 | 空间升级需要特定资源 | 种田空间/炼丹空间/藏书空间 |
| 重生+异能 | 重生并觉醒异能 | 异能使用有代价 | 预知/透视/心灵感应 |

## 默认升级体系

都市重生的"升级"不是修炼，而是社会地位的跃升：

```
底层（1-30章）
  → 小有所成（30-80章）
    → 行业新贵（80-150章）
      → 商业巨头（150-250章）
        → 幕后大佬（250-400章）
          → 传奇人物（400+章）
```

**跃升设计要点**：
- 每次跃升依靠"先知优势+个人努力+关键选择"
- 蝴蝶效应：越往后，前世记忆越不靠谱
- 人脉积累：每个阶段结交的盟友是下阶段的助力

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 重生适应 | 1-5章 | 重生回XX年，适应重生身份 | 铺垫为主，展示前世遗憾 |
| 第一桶金 | 5-20章 | 利用先知优势赚到第一桶金 | 爽点密集 |
| 树敌 | 20-40章 | 成功引起注意，被人针对 | 节奏放缓，制造冲突 |
| 反击 | 40-60章 | 利用前世经验化解危机 | 高潮 |
| 升维 | 60-100章 | 进入更大舞台 | 铺垫+爽点交替 |
| 蝴蝶效应 | 100+章 | 未来开始偏移，先知优势减弱 | 增加不确定性，制造新冲突 |

## 开篇钩子推荐

### 钩子1：遗憾重生（经典）
> 前世重大遗憾 → 重生到关键节点 → 这一次绝不重蹈覆辙

### 钩子2：危机重生（紧张）
> 重生到危机时刻 → 必须立刻做出关键选择 → 选对了翻盘，选错了重来？

### 钩子3：信息差（新颖）
> 重生后发现某些前世不知道的信息 → 一切可能不是前世看到的那样 → 重新审视人生

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 开挂太强 | 先知优势太大，没有挑战 | 蝴蝶效应逐渐削弱先知优势 |
| 年代感失真 | 重生到90年代但写出了2020年的感觉 | 研究目标年代的社会环境 |
| 抄袭现实 | 过度模仿真实商业案例 | 以现实为参考但创造虚构事件 |
| 感情线生硬 | 所有女性角色都爱上主角 | 给每个角色独立的人格和动机 |
