# 修仙体系 · 类型模板

> "大道三千，我只取一条。"

## 类型定位

- **核心卖点**：修仙证道+境界突破+天人感悟+长生之念
- **读者画像**：18-35岁男性，偏好硬核修炼逻辑与内心悟道，享受循序渐进的沉浸感
- **创作难度**：⭐⭐⭐⭐（境界体系设计是核心，感悟描写要有深度，长线不崩极难）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 灵根变异 | 觉醒变异灵根，修炼速度倍增 | 变异灵根难以驾驭，容易走火入魔 | 天灵根/混沌灵根/阴阳灵根 |
| 传承秘境 | 进入独立秘境空间修炼，时间流速不同 | 秘境有进入次数和时长限制 | 洞天福地/上古秘境/随身道场 |
| 道韵残片 | 获得上古大能的道韵残片，辅助悟道 | 残片需同属性功法激活，领悟因人而异 | 剑道残片/丹道残片/阵道残片 |
| 因果之线 | 可窥探因果，趋吉避凶 | 窥探因果要承受反噬，改因果代价更大 | 天眼/命盘/因果镜 |

## 默认升级体系

修仙体系的核心是"境"与"悟"的结合，每个大境界不仅是力量的跃升，更是对天地规则理解的深化：

```
练气期（1-40章）· 感应天地灵气
  → 筑基期（40-100章）· 凝聚灵力根基
    → 结丹期（100-180章）· 丹成一体，寿元大增
      → 元婴期（180-300章）· 神魂蜕变，可夺舍重生
        → 化神期（300-450章）· 神识外放，初步掌控规则
          → 炼虚期（450-600章）· 虚实转换，遁入虚空
            → 合体期（600-800章）· 天人合一
              → 大乘期（800-1000章）· 大道将成
                → 渡劫期（1000+章）· 天劫洗礼，飞升在即
```

**突破设计要点**：
- 练气→筑基靠积累，筑基→结丹靠悟性，结丹以后每一步都是对大道的理解深化
- 每个大境界设初期/中期/后期/大圆满四个小境界
- "瓶颈"是修仙文的灵魂——不是打怪升级能解决的，需要"悟"
- 天劫设计要因人而异，反映修士自身的道与因果
- 境界之间的质变要有鲜明的表现差异（如元婴出窍、化神御天）

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 入道期 | 1-10章 | 展示凡人/低阶修士的局限，接触修仙之门 | 铺垫为主，建立世界观 |
| 练气筑基 | 10-50章 | 初入修仙界，寻功法觅灵石，第一次斗法 | 稳步推进，小爽点不断 |
| 宗门历练 | 50-100章 | 宗门大比/秘境探险，崭露头角 | 爽点密集，结交道友 |
| 结丹悟道 | 100-180章 | 寻找结丹契机，经历生死感悟 | 节奏放缓，重在内心描写 |
| 宗门变故 | 180-250章 | 宗门遭难/被逐/主动离开，独闯修仙界 | 冲突升级，从安逸转向险恶 |
| 元婴蜕变 | 250-350章 | 独自历练，心魔劫难，突破元婴 | 沉淀+爆发的交替 |
| 大道之争 | 350+章 | 与顶尖强者论道斗法，争夺大道机缘 | 宏大叙事，格局拉满 |

## 开篇钩子推荐

### 钩子1：凡人入道（经典）
> 展示凡人世界的局限与苦难 → 偶然接触修仙 → 知道长生可期，决心入道

### 钩子2：废灵根逆天（热血）
> 被判定为废灵根无法修仙 → 发现变异灵根/特殊机缘 → 以残破之身踏上修仙路

### 钩子3：道心叩问（深沉）
> 开篇就是一场关于"道"的追问 → 主角对生命/天道/长生有独特理解 → 以悟性弥补资质不足

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 境界注水 | 突破描写千篇一律，只有"光芒大盛" | 每次突破融入对道的感悟，突破即悟道 |
| 修仙无仙味 | 写成了换皮玄幻，没有修仙的独特气质 | 多写"清修""问道""心魔""因果"等修仙独有元素 |
| 丹药通胀 | 后期丹药泛滥，修仙变成了嗑药 | 限制丹药效果，丹药只能辅助不能替代悟道 |
| 寿元设定废 | 人人活千年万年，寿元毫无压力感 | 寿元是修仙文的核心矛盾之一，要持续强调 |
| 感悟空洞 | 悟道描写只有"顿悟"二字 | 用具体情境和人生经历支撑感悟，悟即成长 |
