# 末世生存 · 类型模板

> "文明崩塌之后，活着就是最大的奢侈。"

## 类型定位

- **核心卖点**：极端生存+资源争夺+人性考验+重建秩序
- **读者画像**：18-35岁，喜欢高压生存和人性博弈，享受"末世之中谁是盟友谁是敌人"的紧张感
- **创作难度**：⭐⭐⭐⭐（生存压力的持续性很难维持，人性描写需要深度）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 异能觉醒 | 末世中觉醒超自然能力 | 能力使用消耗体力/精神力，过度使用会失控 | 元素系/强化系/精神系/特殊系 |
| 空间储备 | 拥有随身储物空间 | 空间有容量限制，且不能存放活物 | 小型空间/农场空间/制造空间 |
| 末世预知 | 提前知道末世爆发的时间 | 只知道大方向，细节不可预知 | 重生预知/系统预警/直觉感知 |
| 进化体质 | 身体在末世环境中加速进化 | 进化方向不可控，可能偏向非人化 | 丧尸抗体/辐射适应/异兽血脉 |

## 默认升级体系

末世的"升级"是生存等级的跃升，核心指标是安全区等级和团队实力：

```
挣扎求生（1-30章）· 朝不保夕，随时可能死亡
  → 小型据点（30-80章）· 拥有基本庇护和食物来源
    → 安全社区（80-150章）· 建立稳定防御和补给体系
      → 基地势力（150-250章）· 区域性力量，开始争夺资源
        → 联盟核心（250-400章）· 多方势力中的关键角色
          → 文明重建（400+章）· 建立新秩序，探索末世真相
```

**升级设计要点**：
- 每个阶段的核心矛盾不同：求生→资源→地盘→秩序→真相
- 生存压力始终存在，不能因为升级就完全消除
- 人性考验贯穿始终：盟友可能背叛，敌人可能合作
- 末世环境的威胁要持续升级（丧尸变异/天灾加剧/资源枯竭）

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 末世降临 | 1-5章 | 灾难爆发，文明崩溃 | 节奏极快，冲击力拉满 |
| 绝境求生 | 5-20章 | 在混乱中寻找庇护和补给 | 高压持续，小喘息 |
| 结伴同行 | 20-40章 | 遇到同伴，组建小队 | 节奏放缓，建立情感 |
| 据点之争 | 40-70章 | 争夺安全据点，与其他幸存者冲突 | 冲突升级，人性博弈 |
| 势力扩张 | 70-120章 | 建立基地，发展势力 | 建设与战斗交替 |
| 末世真相 | 120+章 | 探索末世成因，寻找解法 | 格局拉大，悬念深化 |

## 开篇钩子推荐

### 钩子1：倒计时（经典）
> 末世爆发前几小时/几天 → 主角知道灾难将至 → 在最后时间内做准备

### 钩子2：废墟醒来（紧张）
> 主角在末世废墟中醒来 → 身边是残骸和变异体 → 第一个生存选择就是生死

### 钩子3：抉择时刻（人性）
> 主角面临"救自己还是救别人"的抉择 → 不同的选择导向不同的末世之路 → 末世第一课：没有对错

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 生存无压力 | 主角很快就不愁吃穿，末世变旅游 | 生存物资始终紧张，安全感永远不完整 |
| 人性脸谱化 | 坏人纯坏，好人纯好 | 末世中每个人都有自己的立场和苦衷 |
| 变异体无进化 | 丧尸/异兽永远同一水平 | 威胁要随时间进化，保持压迫感 |
| 基地建设无聊 | 大段描写基建细节 | 基建要有危机驱动，建一半来袭击 |
| 后期变玄幻 | 从生存文变成了打怪升级 | 始终保持生存的底色，能力是生存手段不是碾压工具 |
