# 游戏竞技 · 类型模板

> "王者不是天生的，是一局一局打出来的。"

## 类型定位

- **核心卖点**：技术碾压+逆风翻盘+热血团队+竞技荣耀
- **读者画像**：16-30岁，有游戏经历，喜欢"手速碾压""绝地翻盘"的爽感
- **创作难度**：⭐⭐⭐（比赛节奏容易重复，长期连载要不断出新）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 超级反应 | 超人的手速和反应速度 | 身体负荷大，过度使用会受伤 | 神经加速/肌肉记忆/预判直觉 |
| 游戏天赋 | 对特定游戏有超凡的天赋理解 | 天赋只对一类游戏有效，换游戏要重新适应 | 微操天赋/大局观天赋/创新天赋 |
| 数据分析 | 可实时分析对手行为模式 | 分析需要时间，实时战斗中不总是有用 | 数据面板/行为预测/弱点扫描 |
| 回到巅峰 | 重生/穿越到职业生涯早期 | 记忆模糊且版本已变，老经验未必适用 | 意识重生/记忆回溯/身体穿越 |

## 默认升级体系

游戏竞技的"升级"是竞技层级的攀升，从路人到传奇：

```
路人局（1-20章）· 普通玩家水平
  → 天梯前列（20-50章）· 服务器顶尖
    → 职业试训（50-80章）· 进入职业战队视野
      → 职业新秀（80-130章）· 职业赛场上崭露头角
        → 一线选手（130-200章）· 联赛顶尖选手
          → 世界级（200-300章）· 国际赛场上争锋
            → 传奇封神（300+章）· 被载入电竞史册
```

**升级设计要点**：
- 每个层级对应不同的对手强度和比赛压力
- 从个人操作到团队配合，核心能力逐步转变
- 版本更迭：游戏版本变化，强者适应，弱者淘汰
- 伤病与心理：职业生涯的隐形敌人

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 展露天赋 | 1-10章 | 在路人局中展露惊人天赋 | 快速建立主角实力印象 |
| 一战成名 | 10-25章 | 关键比赛/排位赛一战成名 | 爽点密集 |
| 入队试炼 | 25-45章 | 加入/试训职业战队 | 新环境适应+人际冲突 |
| 联赛初赛 | 45-80章 | 正式参加职业联赛 | 比赛节奏，胜负交替 |
| 季后赛 | 80-120章 | 进入季后赛，对手更强 | 高压叙事，战术博弈 |
| 总决赛 | 120-160章 | 联赛决赛/国际赛事 | 最高潮，所有伏笔收束 |
| 新赛季 | 160+章 | 版本更新/队伍变动/新挑战 | 重置压力，开启新循环 |

## 开篇钩子推荐

### 钩子1：王者归来（经典）
> 曾经的顶尖选手因故离开 → 重回赛场 → 用实力证明自己还是那个王者

### 钩子2：天才初现（热血）
> 一个完全没玩过游戏的"白纸" → 第一次接触就展现恐怖天赋 → 所有人都惊了

### 钩子3：被迫上场（紧张）
> 主角因意外被推上赛场 → 所有人都不看好 → 绝境中逆天操作翻盘

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 比赛重复 | 每场比赛的写法和节奏一样 | 每场比赛有不同的战术焦点和情感驱动 |
| 纯操作描写 | 只有"手速多快""操作多秀" | 操作要服务于战术和人物，写关键时刻的决策 |
| 团队脸谱化 | 队友只是背景板 | 给每个队友独特的游戏风格和性格，团队配合要有化学反应 |
| 忽视版本 | 游戏版本永远不变 | 版本更迭制造新的挑战，也创造新的机会 |
| 无脑碾压 | 主角每场都赢 | 有选择地输，失败是成长的催化剂，也制造悬念 |
| 电竞变言情 | 感情线喧宾夺主 | 感情线服务于人物塑造，不能压过竞技主线 |
