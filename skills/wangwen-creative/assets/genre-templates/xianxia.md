# 东方玄幻 · 类型模板

> "一剑破万法，一步踏山河。"

## 类型定位

- **核心卖点**：热血升级+宏大世界+逆天改命
- **读者画像**：18-30岁男性，追求爽感和代入感，碎片化阅读场景
- **创作难度**：⭐⭐⭐（入门容易，写好难，长篇不崩更难）

## 推荐金手指

| 类型 | 金手指设计 | 限制机制 | 适用变体 |
|---|---|---|---|
| 神秘血脉 | 觉醒远古血脉，逐步解锁血脉神通 | 血脉觉醒伴随痛苦，每次觉醒需特殊契机 | 龙血/凤血/魔血/圣血 |
| 传承法宝 | 获得上古强者传承的法宝/功法 | 法宝需特定条件才能激活下一层 | 戒指/玉佩/残卷/塔 |
| 特殊体质 | 万中无一的修炼体质 | 体质觉醒前被视为废物 | 霸体/圣体/混沌体/不死体 |
| 气运之子 | 天生大气运，机缘不断 | 气运有起伏，低谷期极其危险 | 天命/逆命/窃运 |

## 默认升级体系

```
炼体境（1-30章）
  → 聚气境（30-80章）
    → 筑基境（80-150章）
      → 金丹境（150-250章）
        → 元婴境（250-400章）
          → 化神境（400-600章）
            → 合体境（600-800章）
              → 大乘境（800-1000章）
                → 渡劫境（1000+章）
```

**突破设计要点**：
- 每个大境界分为初期/中期/后期/巅峰四个小境界
- 越往后突破越难，需要契机（天材地宝/顿悟/生死考验）
- 跨越大境界要有质变表现（不仅是数值提升，要有新能力）

## 典型节奏模板

| 阶段 | 章节范围 | 核心事件 | 节奏 |
|---|---|---|---|
| 废物期 | 1-5章 | 展示主角的"废物"状态+金手指初现 | 铺垫为主，埋悬念 |
| 觉醒期 | 5-15章 | 金手指激活，第一次展示实力 | 节奏加快，第一个爽点 |
| 扬名期 | 15-30章 | 在小范围崭露头角，招惹势力 | 爽点密集 |
| 危机期 | 30-50章 | 遭遇强大势力压迫，被迫离开 | 节奏放缓，埋伏笔 |
| 成长期 | 50-80章 | 在新环境修炼升级，结交盟友 | 铺垫+小爽点交替 |
| 爆发期 | 80-100章 | 回归/大比/危机解除，实力碾压 | 高潮密集 |

## 开篇钩子推荐

### 钩子1：废物+奇遇（经典）
> 开篇展示主角被看不起 → 偶然获得金手指 → 第一次展示实力

### 钩子2：危机+觉醒（紧张）
> 开篇主角面临生死危机 → 临界觉醒金手指 → 化险为夷但引出更大麻烦

### 钩子3：反差+悬念（新颖）
> 开篇展示主角的某种反差（表面废物实则隐藏实力） → 通过侧面暗示引发好奇 → 逐步揭示

## 常见坑点与避坑

| 坑点 | 表现 | 避坑方法 |
|---|---|---|
| 战力崩坏 | 后期战力膨胀，前期设定失效 | 每个等级锚定具体表现，不超过10级主线 |
| 无脑打脸 | 每个场景都是被看不起→展示→震惊 | 打脸要有铺垫，对手的嚣张要有理由 |
| 升级太顺 | 主角没有挫折一路碾压 | 每3次升级安排1次受挫，挫折要有叙事功能 |
| 配角工具人 | 配角出现即消失 | 给重要配角独立的故事线 |
| 世界观空转 | 地图换了一个又一个但本质相同 | 每个新区域有独特的规则和冲突 |
