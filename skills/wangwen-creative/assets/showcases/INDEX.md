# 类型模板展示索引

## 可用模板

| 类型 | 模板文件 | 核心卖点 | 适合人群 |
|------|---------|---------|---------|
| 东方玄幻 | `genre-templates/xianxia.md` | 热血升级+宏大世界+逆天改命 | 追求爽感和代入感 |
| 修仙体系 | `genre-templates/cultivation.md` | 修仙悟道+因果业力+超脱 | 偏好修炼体系深度 |
| 都市重生 | `genre-templates/urban-rebirth.md` | 先知优势+逆袭改命 | 有生活阅历的读者 |
| 系统升级流 | `genre-templates/system-flow.md` | 系统辅助+数据化成长 | 喜欢游戏化体验 |
| 无限流 | `genre-templates/infinite-flow.md` | 副本挑战+规则博弈 | 喜欢智斗和生存 |
| 悬疑探案 | `genre-templates/suspense.md` | 逻辑推理+悬念反转 | 喜欢烧脑推理 |
| 末世生存 | `genre-templates/apocalypse.md` | 生存压力+人性考验 | 喜欢紧张刺激 |
| 科幻未来 | `genre-templates/scifi-future.md` | 科技想象+文明演进 | 科幻爱好者 |
| 历史穿越 | `genre-templates/history-time-travel.md` | 历史知识+政治博弈 | 历史爱好者 |
| 游戏竞技 | `genre-templates/game-esports.md` | 技术成长+竞技对抗 | 游戏玩家 |

## 使用方式

1. 创作方向顾问推荐类型后，Read对应模板
2. 基于模板的推荐金手指/升级体系/节奏模板进行定制
3. 将定制后的设定写入 `novel-spec.md`
