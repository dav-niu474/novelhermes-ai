## 五大核心能力

### ⬇️ 小说搜索下载

在番茄小说平台搜索、定位、下载目标作品。支持书名/作者/URL/关键词搜索，获取书籍信息、章节目录、批量下载章节内容。下载与拆文自动衔接，学完即删，尊重版权。

### 📖 拆文分析

把热门网文拆成可复用的创作零件。六维度框架：结构/角色/设定/节奏/文笔/商业。原理提取三层次（表层技巧→中层原理→深层哲学），每篇拆文输出 ≥5 个可复用零件，每个 ≥8 分。

### 🔄 场景全流程拆文仿写

搜索下载→精读标注→六维度拆解→原理提取→仿写创作→对比验证+素材入库的一体化工作流。5种场景级仿写模板（开篇/爽点/伏笔/对话/反转），三步转化法（模式识别→参数调优→创意注入），"三换原则"验证原创性。

### 🗄️ 素材库积累

15类素材分类体系（金手指/升级体系/世界观/人物模板/爽点配方/伏笔模式/节奏模板/冲突模板/对话技巧/描写技巧/转折技法/开头模板/结尾模板/名场面/语言风格），质量评分入库，支持多维检索。

### ✍️ 网文创作

从大纲到正文的完整创作流程。15种类型模板（玄幻修仙/都市重生/系统流/无限流/悬疑推理/历史军事/科幻末世/游戏竞技等），伏笔追踪系统，节奏三定律，5维度评审。

## 核心机制

### 小说搜索下载

番茄小说平台搜索 → 书籍信息获取 → 章节目录提取 → 批量下载 → 本地存档。下载范围按拆文目的智能匹配，仅用于学习。

### 拆文六维度框架

| 维度 | 拆解内容 | 可复用产出 |
|------|---------|-----------|
| 结构 | 开篇钩子/卷结构/剧情螺旋/转折设计 | 结构模板 |
| 角色 | 主角DNA/反派镜像/关系网/对话辨识度 | 人物模板 |
| 设定 | 核心矛盾/力量体系/势力格局/揭示节奏 | 世界观框架 |
| 节奏 | 爽点节奏图/伏笔管理/钩子分析/呼吸图 | 节奏模板 |
| 文笔 | 叙事技法/名场面/对话技法/描写技法 | 风格配方 |
| 商业 | 市场定位/读者心理/商业节奏 | 商业策略 |

### 场景全流程拆文仿写

搜索下载 → 精读标注 → 六维度拆解 → 原理提取 → 仿写创作 → 对比验证+入库。5种场景仿写模板（开篇/爽点/伏笔/对话/反转），"三换原则"验证原创性。

### 素材库体系

15类素材 × 4维评分（独特性30% + 可操作性30% + 扩展性20% + 类型契合度20%）≥7分入库。

### 创作节奏三定律

- **3000字定律**：每3000字至少一个小钩子
- **3章定律**：每3章至少一个大爽点
- **10章定律**：每10章完成一个小剧情弧

### 伏笔追踪系统

三级预警：主线30章/支线20章/细节10章未回收触发预警，防烂尾保险。

### 5维度评审

类型契合度 · 节奏把控 · 人物塑造 · 世界观自洽 · 可读性，各0-10分，输出Keep/Fix/Quick Wins。
